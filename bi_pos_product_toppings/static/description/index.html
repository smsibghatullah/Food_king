POS Product Topping Odoo App

Point of Sales Product Modifier Odoo Apps

POS Pizza Maker Odoo Apps

POS Product Topping Modifier odoo app will help the fast food businesses where the snacks toppings are there. Users can create different toppings for products, create topping groups with different toppings, add topping groups to products, the same products can have different toppings, and also mass update toppings for products. Users have the option to select and add toppings from the pos screen and the toppings price will be added to the main product, user also has the option to automatically add all toppings to the cart that are added to the main products and printed to the pos receipt.

Features

Create Product Toppings

Users can create different toppings for the product.

Create Topping Groups

User can create multiple type of groups for toppings.

Add Toppings to Products

Users can add topping groups and toppings will automatically be added to the product.

Mass Update Product Toppings

Users can also mass updated product toppings for multiple products with a single click.

Auto Add Product Toppings to POS Cart

Added option to automatically add all toppings added to product into pos cart.

Print POS Receipt with Toppings

Added toppings will also printed on pos order receipts.

Create Product Toppings

UUnder the product variant, the user can create product toppings and enable the "Is Topping" option.

Toppings Groups Menu

Under Point of Sale > Configuration > Toppings Groups menu user can create multiple topping groups with different topping items.

Create Toppings Groups

Add Toppings For POS Category

Users can also add toppings for pos category.

Add Toppings on Product

User can also add toppings on product variants under the toppings tab. If the user selects a topping group then added topping to group is automatically added.

Mass Update Toppings

User also has the option to mass update toppings for multiple products with a single click.

User can select or remove topping groups and toppings.

Updated Toppings

Users can see toppings are updated for selected products.

Update Toppings Mannualy

Users can also update toppings manually.

POS Toppings Configuration

User need to check the "Enable Toppings" option to use add topping in pos products.

Add Toppings to POS Cart

Users can see the "Toppings" button on pos screen.

When the user clicks the "Toppings" button user can see a new popup with all added toppings on that product.

Users can click on specific toppings and this will be added to the pos cart, User can also see the toppings price also added to main product.

Remove Toppings

Users can also remove toppings by clicking the "Delete" icon on toppings.

Toppings on POS Receipt

Users can see added toppings also printed on pos receipts.

Auto Add Product Toppings

If a user enables the "Add toppings on product add" option then added all toppings to the product are automatically added to the pos cart on selecting that product, User also can select multiple products with multiple toppings.

Users can see on the selection of specific products all added toppings to that product automatically added to the pos cart.

Users can see printed pos order receipts.
